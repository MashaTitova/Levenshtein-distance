# Levenshtein-distance
## Программа для вычисления расстояния Левенштейна между двумя строками:
- Рекурсивным методом 
- Методом динамического программирования
- Сравнивается время работы двух методов на одинаковых наборах данных
### Входные данные:
- Две строки длиной до 100 символов
### Как запустить через Visual Studio
- Склонируйте репозиторий 
- Откройте файл Lab_3.var26.sln
- Запустите программу
### Как склонировать репозиторий
- Скопируйте ссылку на репозиторий с GitHub. Ссылка: https://github.com/MashaTitova/Levenshtein-distance.git
- В терминале введите команду git clone <ссылка на репозиторий>
- После выполнения команды в текущей папке появится папка с тем же именем, что и сам репозиторий. Если нужно указать другую папку, в конце команды для клонирования нужно указать её название: git clone <ссылка на репозиторий> <папка>
### Пример использования
- Пользователь вводит две строки в соответствующие поля
- При нажатии на кнопку расчитать на листе элемента TabControl для рекурсиного метода производится вычисление расстояния Левенштейна рекурсивным методом
- При нажатии на кнопку расчитать на листе элемента TabControl для метода динамического программирования производится вычисление расстояния Левенштейна метода динамического программирования
- На соответствующем листе элемента TabControl модно увидеть таблицу сравнения результатов работы двух методов
- На форме и каждом листе TabControl расположена кнопка для вызова справки пользователя
### Особенности программы
- Полная валидация ввода (проверка на число, пустой ввод)
- Графический интерфейс
- Визуальное отображение результата
### Требования
- .NET 4.8 или новее
- Visual Studio 2022 (или любая среда с поддержкой C#. Версия языка 14.0 и выше)
